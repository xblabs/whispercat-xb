WhisperCat Portable v1.6.0-xblabs
===================================

Thank you for using WhisperCat!

REQUIREMENTS:
-------------
- Java 11 or higher must be installed
- Download Java from: https://adoptium.net/

HOW TO RUN:
-----------
Simply double-click WhisperCat.exe

TO PIN TO TASKBAR:
------------------
1. Right-click on WhisperCat.exe
2. Select "Pin to taskbar"
3. You can now launch WhisperCat from your taskbar anytime!

SETTINGS:
---------
Your settings are automatically saved to:
%APPDATA%\WhisperCat\config.properties

This includes:
- API keys
- Global hotkeys
- Microphone preferences
- Post-processing pipelines
- All other settings

Your settings will persist even if you move WhisperCat.exe to a different location!

FEATURES:
---------
- Record audio from system microphone
- Transcribe using OpenAI Whisper, FasterWhisper, or OpenWebUI
- Global hotkey support (activate from anywhere!)
- Post-processing pipelines for text transformation
- Drag & drop audio files
- Multiple theme options

TROUBLESHOOTING:
----------------
If WhisperCat won't start:
1. Make sure Java 11+ is installed (run "java -version" in Command Prompt)
2. Install Java from https://adoptium.net/ if needed
3. Right-click WhisperCat.exe and select "Run as administrator" (if needed)
